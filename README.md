# FLIP Dashboard

Интерактивный дашборд для расчёта экономики флип-сделок по Москве: покупка, ремонт, продажа, налоги, стресс-сценарии и годовая доходность.

## Что внутри

- Финмодель сделки с ключевыми KPI (валовая/чистая прибыль, ROI, доходность)
- Воронка сделки и «водопад» структуры прибыли
- Стресс-тест по цене продажи
- Сценарии прибыли по количеству сделок в год
- Табличные матрицы для быстрой оценки порогов входа
- Поддержка загрузки данных ЦИАН (`cian_data.json` / CSV)
- Автодеплой на GitHub Pages через Actions

## Быстрый запуск

### Вариант 1: просто открыть

Откройте `index.html` в браузере.

### Вариант 2: локальный сервер (рекомендуется)

```bash
python3 -m http.server 8080
```

Далее откройте: <http://localhost:8080>

## GitHub Pages

В репозитории уже есть workflow:

- `.github/workflows/pages.yml`

После push в `main` сайт деплоится автоматически (если в Settings включен Pages через GitHub Actions).

## Формат данных ЦИАН

Дашборд пытается автоматически загрузить `cian_data.json` из корня.
Также можно загрузить JSON/CSV через UI.

Рекомендуемые поля (минимум):

- `price` — цена лота
- `area` — площадь (м²)
- `rooms` — комнаты
- `address` / `metro` — локация

> Если структура отличается — адаптация делается в JS-парсере внутри `index.html`.
